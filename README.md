# BITS Pilani Academic Website

This is a local version of the BITS Pilani Academic website that can be run on your computer.

## 🚀 Quick Start

### Option 1: Python Script (Recommended)
1. Make sure Python 3.x is installed on your system
2. Run one of the following commands:

**On macOS/Linux:**
```bash
python3 launch_website.py
```

**On Windows:**
```bash
python launch_website.py
```

### Option 2: Batch/Shell Scripts

**On Windows:**
Double-click `launch_website.bat` or run:
```cmd
launch_website.bat
```

**On macOS/Linux:**
```bash
./launch_website.sh
```

## 🌐 Accessing the Website

Once the server starts:
- The website will automatically open in your default browser
- If it doesn't open automatically, go to: `http://localhost:8000`
- To stop the server, press `Ctrl+C` in the terminal

## 📁 Website Structure

- **index.html** - Main homepage
- **Student_Login.html** - Student login page
- **student/student-dashboard.html** - Student dashboard (after login)
- **Styles/** - CSS files for styling
- **Scripts/** - JavaScript files for functionality
- **images/** - Images and graphics

## 🔧 Troubleshooting

### Port Already in Use
If port 8000 is already in use, the script will automatically try port 8001, 8002, etc.

### Python Not Found
- **Windows**: Download and install Python from [python.org](https://python.org)
- **macOS**: Install using Homebrew: `brew install python3`
- **Linux**: Install using your package manager: `sudo apt install python3`

### File Path Issues
Make sure all files are in the correct directory structure as provided.
